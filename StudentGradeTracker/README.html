Student Grade Tracker

This is a simple Java program that allows you to track student grades that are entered on the users console.

Features

  1. Add a new student and grade
  2. Display list of students and their grades
  3. Calculate average grade for the subject
  4. Search for a student and display their grade
  5. Exit

How to Run the Program

Compile and run the StudentGradeTracker.java file in your IDE.
